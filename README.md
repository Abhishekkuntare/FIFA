# 🚀 Tech Stack

## Frontend
- Next.js 15
- React 19
- TypeScript
- TailwindCSS

## 3D & Animation
- Three.js
- React Three Fiber
- Drei
- Framer Motion
- GSAP
- React Spring

## State & Utilities
- Zustand
- Lenis Smooth Scroll
- Howler.js

## Rendering
- WebGL
- Postprocessing
- Bloom
- Volumetric Effects
